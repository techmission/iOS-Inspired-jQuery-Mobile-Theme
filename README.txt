iOS-Inspired jQuery Mobile Theme
================================

About This Theme
----------------

This theme is a jQuery Mobile theme for Drupal, with an iOS-like
look and feel. It requires the latest version of jQuery, which
may cause conflicts with some Javascript in Drupal and its
contributed modules. Do not use this theme for site admin.

Base of Theme
-------------

The base of this theme is the iOS-inspired jQuery Mobile theme by Tait Brown,
available at https://github.com/taitems/iOS-Inspired-jQuery-Mobile-Theme.
